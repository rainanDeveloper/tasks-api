# Task app

## Description

Simple task application made with [Nest](https://github.com/nestjs/nest)

## License

This app is licensed under [MIT license](LICENSE).

## Installation

You can install the app by running:

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
